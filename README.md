# about.ifuri.com

A static one-page presentation of the **ifURI** ecosystem — what it is, the core
model (`if URI, then flow`), the URI schemes and the "one contract, any language"
connector story. Published to [about.ifuri.com](https://about.ifuri.com).

## Ecosystem
- Site: [ifuri.com](https://ifuri.com) · Docs: [docs.ifuri.com](https://docs.ifuri.com)
- Runtime: [urirun.com](https://urirun.com) · Connectors: [connect.ifuri.com](https://connect.ifuri.com)
- App download: [get.ifuri.com](https://get.ifuri.com) · Node installer: [get.urirun.com](https://get.urirun.com)

## Development
```bash
make serve    # preview locally
make test     # validate the static page
make deploy   # publish to about.ifuri.com (Plesk rsync)
```
